## 二、商人（shangren）

### 陈瑞丰

| 项目 | 内容 |
|------|------|
| 姓名 | 陈瑞丰 |
| 字 | 润生 |
| 出生 | 雍正元年（1723） |
| 年龄 | 三十七（虚岁） |
| 籍贯 | 祖籍福建泉州府晋江县，客居苏州已十二年 |
| 身份 | 丝绸洋货商 |
| 家世 | 出身泉州海商世家。祖父陈士杰曾随商船下南洋，贩运香料、燕窝，积资颇丰。父陈永昌继承家业，兼营丝绸转口，常往返闽苏两地。瑞丰为长子，自幼随父走商，精通闽南话、吴语及几句洋泾浜英语。乾隆十七年（1752）携家眷定居苏州，于阊门开设"瑞丰号"，主营汉府八丝贡缎与闽粤洋货。妻林氏，泉州人，育有一子一女。 |
| 性情 | 精明干练，为人圆融。重信义，善交际，与阊门各行商皆有往来。虽为商人，却颇慕文风，偶作打油诗自娱。 |
| 经济 | 身家殷实，阊门商铺年流水逾万两。在山塘街设有分号，枫桥另有米粮代理。家中雇有账房、伙计十余人。 |
| 行迹 | 常往返于阊门码头与各地商路，深谙钱庄兑付、船行转运之道。此日正于枫桥米行采办粮货，兼看阊门商铺行情。 |

## 三、平民（pingmin）

### 阿桂

| 项目 | 内容 |
|------|------|
| 姓名 | 阿桂（本名周阿桂） |
| 出生 | 乾隆六年（1741） |
| 年龄 | 十九（虚岁） |
| 籍贯 | 苏州府长洲县 |
| 身份 | 面馆帮工 |
| 家世 | 出身城市贫民。父周大牛，原为虎丘乡间佃农，因赋税繁重于乾隆十一年（1746）携家入城谋生，在山塘街"三鲜大面"铺做帮工，后升为掌勺。母赵氏，在同街一户人家做浆洗衣裳的零工。阿桂为长子，下有一妹名小荷，年十二，在邻家绣坊做学徒。全家租住半塘桥畔一间逼仄平房。 |
| 性情 | 质朴爽朗，手脚麻利。不识字，却记性好，爱听人说书讲古。极爱看街头卖艺，每逢庙会社戏必不缺席。 |
| 经济 | 月工钱三百文，勉强糊口。偶尔帮街坊跑腿送物赚些零钱。最大愿望是攒够银子学一门手艺，不再端盘子。 |
| 行迹 | 每日寅时起身和面、熬汤，在面馆帮工。闲时在山塘街一带送货跑腿。逢虎丘庙会则歇工游逛。 |
| 档案 | [pingmin/阿桂.md](pingmin/阿桂.md) |

|------|------|
| 姓名 | 陆小顺 |
| 出生 | 乾隆三年（1738） |
| 年龄 | 二十一（虚岁） |
| 籍贯 | 淮安府山阳县 |
| 身份 | 商铺伙计 |
| 家世 | 出身运河边船户家庭。父陆大江，常年在京杭大运河上跑船运货。母在码头边摆茶水摊贴补家用。小顺自幼随父在船上长大，识得航道水性，却识字不多。十七岁时经人介绍到扬州商人周茂德的商行做伙计，负责押船送货。此番周茂德在苏州阊门新设分号，派小顺随货船南下赴苏，到分号帮工。 |
| 性情 | 忠厚勤快，手脚麻利。不善言辞却肯吃苦，待人诚恳有礼。初到苏州处处新鲜，事事好奇，常因不识路而闹笑话。 |
| 经济 | 在商行做伙计月工钱五百文，包吃住。此次来苏分号帮工，周老板另给了二两银子安家费。 |
| 行迹 | 随货船沿京杭大运河南下，经淮安、扬州、镇江，过浒墅关抵苏州阊门码头。初到姑苏，对城内街巷、风土人情一概不知，需小主指点。 |
